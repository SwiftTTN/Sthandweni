RECOMMENDED APPROACH FOR IMPLEMENTATION:

1. Visit each Wikimedia Commons category above
2. For each athlete, find a good competition or portrait photo
3. Note the filename
4. Create URL: https://commons.wikimedia.org/wiki/Special:FilePath/[filename]
5. Update the JSON "images" array with the new URL

EXAMPLE:
If you find "Hidilyn Diaz 2020.jpg" on Wikimedia Commons:
{
  "type": "Olympic Champion",
  "url": "https://commons.wikimedia.org/wiki/Special:FilePath/Hidilyn_Diaz_2020.jpg",
  "source": "Wikimedia Commons",
  "license": "CC-BY-4.0",
  "caption": "Hidilyn Diaz at Tokyo 2020 Olympics"
}
